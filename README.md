# dotfiles

@okayama-daiki does dotfiles

## Environment

- MacBoor Air M2, 2022 (macOS Tahoe 26)

## Installation

1. Clone this repository

    ```bash
    git clone git@github.com:okayama-daiki/dotfiles.git
    ```

2. Run the installation script

    ```bash
    make all
    ```
